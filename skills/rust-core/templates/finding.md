# Finding Tracking Template

```
Finding ID: SEC-YYYY-NNN
Severity:   [Critical|High|Medium|Low|Info] (CVSS: X.X)
Component:  [file:line or module]
Description:[one-paragraph technical detail]
Impact:     [what an attacker can achieve]
Remediation:[concrete fix with code reference]
Assignee:   [who]
Status:     [Open|In-Progress|Fixed|Verified|Accepted-Risk]
Deadline:   [date]
Verification-of-fix: [test case or scan result confirming remediation]
```
