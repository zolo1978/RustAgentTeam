# 竞品分析模板 & 成功指标

> 本文件包含竞品分析表格模板和业务 KPI 模板，编写 PRD 时直接复制使用。

## 竞品分析模板

```markdown
| 维度 | 我方产品 | 竞品 A (Eagle) | 竞品 B (Pixave) | 竞品 C (Native) |
|------|---------|---------------|----------------|----------------|
| 平台 | Win/Mac/Linux/iOS/Android | Win/Mac | Mac only | Win/Mac |
| 价格 | 免费 | $30 买断 | $15 买断 | $20/年 |
| 离线能力 | 100% 离线 | 需联网激活 | 完全离线 | 需联网 |
| 批量导入 1K 文件 | < 5s (预估) | ~8s | ~12s | ~6s |
| 包大小 | < 25MB | 85MB | 45MB | 60MB |
| 智能标签 | 自定义 | 预设+自定义 | 仅预设 | 预设+自定义 |
| 差异化优势 | 跨平台最广+开源+轻量 | 插件生态 | macOS 原生体验 | AI 自动标签 |
```

## 成功指标 / 业务 KPI

```markdown
| 指标 | 目标值（V1.0 上线 3 个月） | 测量方式 |
|------|---------------------------|----------|
| DAU | 5,000 | 匿名遥测（用户 opt-in） |
| 次日留存率 | > 40% | 首次启动后 24h 再启动 |
| 7 日留存率 | > 25% | 首次启动后 7d 内再启动 |
| 周活跃用户导入量 | > 100 文件/人/周 | 本地统计 |
| NPS | > 40 | 应用内调查（首次使用 7d 后触发） |
| 付费转化（Pro 版） | > 3% | 订阅漏斗 |
| 崩溃率 | < 0.1% 会话 | Sentry / 自建遥测 |
| 商店评分 | > 4.2 星 | App Store / Google Play |
```
