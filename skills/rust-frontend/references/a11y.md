# 无障碍（Accessibility）Checklist

## 键盘导航

- [ ] 所有交互元素可通过 Tab 键聚焦
- [ ] Enter/Space 可激活按钮和链接
- [ ] Escape 可关闭 Modal/Drawer
- [ ] 焦点陷阱（Focus Trap）在 Modal 打开时生效

```typescript
// Modal Focus Trap 示例
function Modal({ open, onClose, children }: ModalProps) {
  const ref = useRef<HTMLDivElement>(null);
  useEffect(() => {
    if (open) {
      const prev = document.activeElement as HTMLElement;
      ref.current?.focus();
      return () => prev?.focus(); // 关闭时恢复焦点
    }
  }, [open]);
  if (!open) return null;
  return (
    <div role="dialog" aria-modal="true" ref={ref} tabIndex={-1}>
      {children}
    </div>
  );
}
```

## ARIA Live Region（动态内容通知）

- [ ] 表单错误使用 `aria-live="polite"` 通知屏幕阅读器
- [ ] 加载状态使用 `aria-busy="true"`
- [ ] Toast/通知使用 `role="status"` 或 `aria-live="assertive"`

```typescript
// 错误通知组件
function FormError({ message }: { message: string | null }) {
  if (!message) return null;
  return <div role="alert" aria-live="polite" className="text-red-600">{message}</div>;
}
```

## 语义化 HTML

- [ ] 使用 `<button>` 而非 `<div onClick>` 处理点击
- [ ] 表单控件有 `<label>` 或 `aria-label`
- [ ] 图片有 `alt` 属性
- [ ] 页面有 `<main>`、`<nav>`、`<header>` 地标元素
